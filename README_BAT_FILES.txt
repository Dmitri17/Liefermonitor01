Инструкции по использованию BAT файлов для Liefermonitor01
========================================================

Созданы следующие BAT файлы для удобного управления проектом:

1. run.bat - Основной файл запуска
   - Проверяет наличие Java
   - Проверяет наличие JAR файла
   - Запускает приложение с подробной диагностикой
   - Показывает понятные сообщения об ошибках

2. start.bat - Быстрый запуск
   - Простой запуск без проверок
   - Для опытных пользователей

3. build.bat - Сборка проекта
   - Проверяет наличие Maven
   - Выполняет mvn clean package
   - Показывает результат сборки

4. rebuild-and-run.bat - Полный цикл
   - Сначала собирает проект
   - Затем запускает приложение

5. start-silent.bat - Запуск без консоли
   - Минимальное черное окно
   - Перенаправляет вывод в null

6. start-hidden.vbs - Полностью скрытый запуск
   - Никаких окон консоли
   - Запускает только GUI приложения

7. start-minimal.bat - Минимальное окно
   - Маленькое зеленое окно
   - Показывает только заголовок

8. start-silent.ps1 - PowerShell скрипт
   - Полностью скрытый запуск
   - Требует PowerShell

ИСПОЛЬЗОВАНИЕ:
==============

Для первого запуска:
1. Дважды кликните на rebuild-and-run.bat
2. Дождитесь завершения сборки
3. Приложение запустится автоматически

Для обычного запуска (после сборки):
1. Дважды кликните на run.bat

Для быстрого запуска:
1. Дважды кликните на start.bat

Для запуска БЕЗ черного окна:
1. start-hidden.vbs - полностью скрытый запуск (рекомендуется)
2. start-silent.bat - минимальное окно
3. start-minimal.bat - маленькое зеленое окно
4. start-silent.ps1 - PowerShell скрипт

ТРЕБОВАНИЯ:
===========
- Java 17 или выше
- Maven (для сборки)
- Доступ к базе данных SQL Server

ПРОБЛЕМЫ:
=========
Если возникают ошибки:
1. Проверьте, что Java установлена: java -version
2. Проверьте, что Maven установлен: mvn -version
3. Проверьте подключение к базе данных
4. Запустите run.bat для подробной диагностики
