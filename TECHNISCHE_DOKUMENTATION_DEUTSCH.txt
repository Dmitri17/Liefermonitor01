BUILD-PROZESS:
==============
1. Maven Clean: mvn clean
2. Kompilierung: mvn compile
3. Tests: mvn test (übersprungen mit -DskipTests)
4. Package: mvn package
5. Assembly: Erstellt JAR mit Dependencies

START-OPTIONEN:
===============
1. start-hidden.vbs - Versteckter Start (empfohlen)
2. run.bat - Mit Diagnose
3. start.bat - Schnellstart
4. start-silent.bat - Minimales Fenster

FEHLERBEHEBUNG:
===============
Problem: "A java Exception has occured"
Lösung: Verwenden Sie JAR mit Dependencies

Problem: Datenbankverbindung fehlgeschlagen
Lösung: Prüfen Sie Netzwerk und Anmeldedaten

Problem: Java nicht gefunden
Lösung: Java 17+ installieren und PATH setzen

Problem: Maven nicht gefunden
Lösung: Maven installieren und PATH setzen
